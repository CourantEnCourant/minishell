## Instructions

### Build

```bash
make        # Build the minishell executable
make clean  # Remove object files
make fclean # Remove object files and the executable
make re     # Rebuild from scratch
```

### Run minishell

```bash
./minishell
```

### Run tests

```bash
mv ./tests/test.norminette ./tests/test.c
make test
./test
```
